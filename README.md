# pytest-do-not-mock

A pytest plugin that prevents mocking of specific classes and functions.

## Installation

```bash
pip install pytest-do-not-mock
```

## Development

```bash
git clone git@github.com:LifeLex/pytest-do-not-mock.git
cd pytest-do-not-mock
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
pytest
```
